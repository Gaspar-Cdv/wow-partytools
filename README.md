# PartyTools

PartyTools is a minimal World of Warcraft addon for fast access to group utilities and world markers.
It is largely inspired by the addon [wMarker](https://www.curseforge.com/wow/addons/wmarker).

## Features

**World Marker Bar:**
- Left-click: Place a world marker.
- Right-click: Remove the world marker.
- Clear button: Remove all world markers.

**Tools Bar:**
- Ready Check: Start a ready check.
- Role Check: Start a role poll.
- Assign tank/healer marker: Macro button to mark the tank with a square, and the healer with a moon. If there are two tanks/healers, the second one will be marked with a circle/star respectively.
- Countdown: Left-click for a 10s pull timer, right-click to cancel.

Frames are draggable and appear at the top left by default.

It also hides the default compact raid frame manager.
